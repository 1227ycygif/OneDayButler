# OneDayButler
One Day Butler - 하루 집사 : VR Cat Simulation(VR 고양이 키우기 시뮬레이션)

## 프로젝트 소개 
가상현실 환경에서 고양이를 돌보고 상호작용하는 시뮬레이션 

## 담당 역할 
- NavMeshAgent 기반 이동 시스템 
- FSM 기반 행동 관리
- 애니메이션 및 사운드 연동

## 기술 스택 
- Unity 
- C# 
- VR (Meta Quest) 
- NavMeshAgent 
- AudioMixer

## 주요 기능 
- 고양이와 VR 상호작용 
- 돌보기 시스템 (먹이, 씻기기 등)
- 고양이 AI 행동 패턴
- 몰입형 VR 환경

## 개발 기간 
2025.09 ~ 2025.09 (2주/게임부트캠프)
